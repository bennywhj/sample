Prerequisites needed for compiling and running this sample code via c++:

- libpng ( available at: http://www.libpng.org/pub/png/libpng.html )
- libpng++ ( available at: http://www.nongnu.org/pngpp/ )
- OpenCV 2.4 ( available at http://www.opencv.org )

On a ubuntu box you can get libpng via apt:

- sudo apt-get install libpng12-dev
- sudo apt-get install libpng++-dev

Linux:

1) Move to root directory of the sample code
2) Type 'cmake .'
3) Type 'make'
4) Run './sample'
